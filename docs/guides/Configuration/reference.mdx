### Block

| Property          | Type             | Required | Default       | Description                                         |
| ----------------- | ---------------- | -------- | ------------- | --------------------------------------------------- |
| `run`             | string           | yes      | N/A           | Exported function name                              |
| `name`            | string           | no       | `run`         | Block identifier (defaults to `run`)                |
| `enabled`         | boolean          | no       | true          | Enable or disable this block                        |
| `input`           | object           | no       | `{}`          | Input passed to the function                        |
| `env`             | object           | no       | `{}`          | Environment variables for this block                |
| `options.timeout` | string \| number | no       | `1s`          | Execution timeout (`1s` or `1000`)                  |
| `options.retries` | number           | no       | `3`           | Number of retry attempts                            |
| `options.backoff` | string           | no       | `exponential` | Retry strategy: `exponential`, `linear`, `squaring` |
