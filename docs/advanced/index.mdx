## Setup

The guides in this section will reference a `setup` function.

This is a function you export in your code. You use it to register plugins, executors, and orchestrators:

<ApiTabs
  js={`export function setup(xgsd) {
    xgsd.use(MyPlugin)

    // or pass instance
    xgsd.use(new MyPlugin())

    xgsd.executor(MyExecutor)
    xgsd.orchestrator(MyOrchestrator)

}`}

/>
